# Wireshark Introduction Lab

## Objective

Aprender a capturar y observar tráfico de red utilizando Wireshark.

## Tool

Wireshark

## Lab Description

Este laboratorio consiste en capturar tráfico de red desde el equipo local y analizar los paquetes generados durante la navegación web.

## Expected Learning

* Identificar protocolos de red
* Comprender la estructura básica de los paquetes
* Familiarizarse con la interfaz de Wireshark
