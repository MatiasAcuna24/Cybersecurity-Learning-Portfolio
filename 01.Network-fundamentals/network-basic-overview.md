# Network Fundamentals Overview

## Objective

Comprender los fundamentos de redes que sirven como base para el análisis de tráfico y diagnóstico de problemas en entornos TI.

## Key Concepts Studied

* Modelo OSI
* Protocolos de red
* Dirección IP
* Puertos de red
* Comunicación entre dispositivos

## OSI Model

El modelo OSI divide la comunicación de red en siete capas:

1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

Cada capa tiene responsabilidades específicas que permiten la comunicación entre sistemas.

## Why This Is Important

Comprender cómo funcionan las redes permite diagnosticar problemas de conectividad, analizar tráfico de red y entender cómo operan muchos ataques informáticos.
